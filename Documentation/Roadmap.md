# Roadmap

## Idea
The idea of the app is to create a usefull space, where users can track their exercises throughout a workout, save information about sets and view statistics.

## Features

### Workout tracking
Users can create a workout session inside the app. The starting time would either be the one that user puts, or the time that the workout was created, if the time wasn't specified.
Inside each workout, users can add the exercises that they are doing.
Each exercise will have the ability to store information about every set of this exercise, including it's repetition quantity and the used weight.

### Statistics
Users will have the ability to track statistics about every exercise. User would be able to see their max weight on each exercise, see their progress, and so on.
